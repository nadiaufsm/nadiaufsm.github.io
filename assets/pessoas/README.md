# Fotos do time

Esta pasta guarda as imagens usadas na seção **Time NADIA**.

Arquivos principais:

- `gabriel-lunardi.jpg`: foto de Gabriel Machado Lunardi.
- `adriano-quiliao.jpg`: foto de Adriano Quilião Oliveira.
- `estudantes-ic.svg`: imagem provisória para estudantes de iniciação científica.
- `pesquisadores-pg.svg`: imagem provisória para pesquisadores de pós-graduação.

Para atualizar o site, substitua os arquivos mantendo o mesmo nome ou edite os caminhos das imagens no arquivo `index.html`.
